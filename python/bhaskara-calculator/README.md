# Bhaskara Calculator

A simple Python program that calculates the roots of a quadratic equation using the Bhaskara formula (quadratic formula).

## Features
* Input coefficients `a`, `b`, and `c`
* Calculate the discriminant (Δ)
* Compute the real roots of the equation
* Display the results in the terminal

## Technologies
* Python 3

## How to Run
Navigate to the project folder and run:
```
python calculator.py
```

## Learning Goals
This project was created to practice:
* mathematical calculations in Python
* user input handling
* conditional logic
* basic algorithm implementation
